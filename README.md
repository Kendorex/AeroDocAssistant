# AeroDoc Assistant  
**ИИ-ассистент по технической документации в авиастроении**

## Описание проекта

**AeroDoc Assistant** — интеллектуальный ассистент на базе современных **LLM (Large Language Models)**, предназначенный для семантического поиска, анализа и диалогового взаимодействия с корпусом технической документации в авиастроительной отрасли.

Проект разработан в рамках хакатона и ориентирован на создание **экспертной системы**, способной:
- понимать профессиональный контекст авиастроения;
- работать со сложными, объёмными и разнородными техническими документами;
- предоставлять **точные, обоснованные ответы** с привязкой к исходным источникам.

## Ключевые возможности

- Загрузка и обработка технической документации (руководства, спецификации, отчёты)
- Семантический поиск по содержанию документов
- Диалоговый интерфейс для работы с документацией на естественном языке
- Контекстно-зависимые ответы с учётом отраслевой терминологии
- Ссылки на фрагменты документов, использованные при формировании ответа
- RAG-архитектура (Retrieval Augmented Generation)

## Архитектура решения

AeroDoc Assistant реализован на основе **RAG-подхода**, который включает:
1. Предварительную обработку и разбиение документов на смысловые фрагменты  
2. Генерацию векторных представлений (эмбеддингов)  
3. Хранение и поиск релевантных фрагментов в векторной базе данных  
4. Формирование контекста
